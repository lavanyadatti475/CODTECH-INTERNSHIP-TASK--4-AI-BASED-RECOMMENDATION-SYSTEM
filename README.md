# 📌 Project Overview

The **AI-Based Recommendation System** is a Java application that recommends products based on user preferences. It demonstrates the fundamental concept of recommendation systems by using predefined sample data to suggest related products.

The application prompts the user to select a product, searches the recommendation dataset, and displays a list of relevant product recommendations. This project showcases the basic principles behind recommendation engines used in modern e-commerce and content platforms.

---

# 🎯 Objective

Develop a Java application that:

* Accepts user preferences as input.
* Recommends related products.
* Demonstrates the working of a recommendation engine.
* Uses sample product data for generating recommendations.
